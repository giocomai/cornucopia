# Show ad campaign structure in a nomnmonl graph

Show ad campaign structure in a nomnmonl graph

## Usage

``` r
cc_nomnoml(ads_df, direction = "right", drop_vertical_bar = FALSE, ...)
```

## Arguments

- direction:

  Defaults to "right". Can be "down". Passed to \`nomnoml\`.

- drop_vertical_bar:

  Defaults to FALSE. If TRUE, replaces vertical bar (which in
  \`nomnoml\` can be used to separate a block in subsections) with "-".

- ...:
