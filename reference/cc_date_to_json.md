# Output date in a format that can be passed as custom date to the Facebok API.

Output date in a format that can be passed as custom date to the Facebok
API.

## Usage

``` r
cc_date_to_json(start_date, end_date = NULL)
```

## Arguments

- start_date:

  A date.

- end_date:

  Defaults to NULL. If left to NULL, \`end_date\` is assumed to be the
  same as the start date
